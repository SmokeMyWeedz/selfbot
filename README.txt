Si vous n'avez pas le dossier 'node_modules' allumez cmd, sélectionner le chemin d'accès de votre
self bot avec "cd (chemin d'accès)",
après lance le.exe si discord se ferme c'est que cela a marché relancé le 
ensuite tapez "npm install --save discord.js" (il faut avoir installé nodejs avec npm)
Et ensuite pour démarrer le bot tapez "npm start"
Et pour l'éteindre tu fais CTRL + C
